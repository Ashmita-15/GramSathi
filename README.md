## 🚀 **Overview**  
**GramSathi** is a multilingual telemedicine web + mobile application designed to bring quality healthcare access to rural and remote communities. It enables patients to consult doctors, maintain offline health records, and receive AI-driven first-aid assistance — even in low-bandwidth environments.

---

## 🌟 **Key Features**  

- 🌐 **Multilingual Support** – Accessible to users in their preferred language.  
- 🩺 **Teleconsultation** – Schedule and book video consultations with doctors.  
- 📤 **Offline Symptom Uploads** – Patients can upload audio, video, or documents to describe symptoms in case of poor connectivity.  
- 📝 **Offline Health Records** – Store patient health data locally for rural use.  
- 🤖 **AI Symptom Checker** – Provides first aid, prescription reading, medicine explanations, and basic report analysis offline.  
- 💊 **Medicine Availability Updates** – Get real-time updates from nearby pharmacies.  
- 🛍️ **E-Pharmacy Integration** – Order medicines directly through the platform.  
- 🚨 **Real-Time SOS Alerts** – Sends emergency alerts to nearby hospitals during critical situations.

---

## 🛠️ **Tech Stack**  
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js / Express  
- **Mobile:** React Native (for cross-platform access)  
- **Database:** MongoDB  
- **AI / ML:** Custom symptom checker models  
- **Deployment:** Render / Vercel / Netlify / Heroku

---

## 💡 **Vision**  
GramSathi aims to **make healthcare accessible and inclusive** for everyone, regardless of their location or internet connectivity. It’s designed especially for rural communities that often face challenges in reaching quality healthcare facilities.
